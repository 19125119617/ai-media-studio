## AI 创作工坊 SaaS 网站 - 视频 API 修复

**问题**：图生视频报错 `Field required: input.media`，文生视频可能也有类似问题

**根因**：百炼 wan2.7 新版 API 协议变更：
- 模型名：`wan2.7-t2v-2026-04-25` / `wan2.7-i2v-2026-04-25`（不是旧的 `wan2.7-t2v` / `wan2.7-i2v`）
- 文生视频：`input.prompt`（不是 `input.text`），parameters 加 `ratio`
- 图生视频：`input.media[{type:'first_frame', url:'...'}]`（不是 `input.image_url`），`input.prompt` 代替 `input.text`

**修复**：
- `backend/server.js`：t2v 和 i2v 接口全部改用新版字段结构，从 `plan.params.modelName` 读取模型名（fallback 到正确值）
- `backend/config.json`：t2v/i2v 4个模型 params 增加 `modelName`
- `frontend/public/index.html`：t2v 面板新增尺寸比例选择器（16:9/9:16/1:1/4:3/3:4），`selectedRatio` 初始化加 `t2v`，`doGenerate` 传 `ratio` 参数
- `selectRatio` 函数改为通用，支持任意 tab
