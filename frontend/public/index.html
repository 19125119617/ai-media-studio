API 接口文档

所有接口均兼容 OpenAI 格式，使用 Bearer Token 认证

GET /api/models
获取所有已启用的模型列表，包含模型 ID、类型、方案标签等信息
GET /api/models/pricing
获取模型定价信息，包含所有模型的价格、类型和服务商（无需认证）
POST /api/generate/text-to-image
文生图接口，支持 wan2.7、wanx2.0、DALL-E 等模型，可选尺寸比例和数量
POST /api/generate/text-to-video
文生视频接口，支持 wan2.7 和欢乐马系列模型，可选分辨率和时长
POST /api/generate/image-to-video
图生视频接口，上传首帧图片生成动态视频，支持 wan2.7 和欢乐马系列
GET /api/task/:taskId
查询生成任务状态和结果，任务异步执行需轮询获取最终结果
